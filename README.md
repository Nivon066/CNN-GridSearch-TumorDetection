# CNN-GridSearch-TumorDetection
Clasificación de tumores cerebrales en MRI mediante CNN y Grid Search. Incluye optimización de hiperparámetros, métricas de reproducibilidad (semillas 1337/123), uso de GPU NVIDIA A100 y validación con Early Stopping. Modelo óptimo logrado con 96.98% de exactitud y 98.88% de AUC.
